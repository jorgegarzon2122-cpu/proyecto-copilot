1. GRAFO NO DIRIGIDO SIMPLE
================================================================================

      A
     / \
    /   \
   B     C
    \   /
     \ /
      D

Representacion:
- A -- B
- A -- C
- B -- D
- C -- D

Lista de Adyacencia:
A: [B, C]
B: [A, D]
C: [A, D]
D: [B, C]


2. GRAFO DIRIGIDO (DIGRAFO)
================================================================================

      A
      |
      v
      B ---> C
      |      ^
      |      |
      v      |
      D -----+

Aristas dirigidas:
- A -> B
- B -> C
- B -> D
- D -> C


3. GRAFO PONDERADO
================================================================================

      A
     /5\
    /   \
   B-3--C
   |     |
   4     2
   |     |
   D-1--E

Pesos en aristas:
A-B: 5
A-C: 3
B-D: 4
C-D: 2
D-E: 1
C-E: 2 (no mostrado para claridad)


4. ÁRBOL (Grafo acíclico conexo)
================================================================================

          A
        / | \
       /  |  \
      B   C   D
     /|      / \
    E F     G   H

Propiedades:
- No tiene ciclos
- Conexo
- n vertices = n-1 aristas


5. MATRIZ DE ADYACENCIA
================================================================================

Para el grafo: A-B, A-C, B-D, C-D

    A  B  C  D
A [ 0  1  1  0 ]
B [ 1  0  0  1 ]
C [ 1  0  0  1 ]
D [ 0  1  1  0 ]

Nota: Simetrica para grafos no dirigidos


6. BÚSQUEDA DFS (Depth-First Search)
================================================================================

Grafo:
    A
   / \
  B   C
  |   |
  D   E

Orden DFS desde A: A -> B -> D -> C -> E
(Explora profundamente antes de retroceder)

Stack (pila):
Paso 1: [A]
Paso 2: [B] (pop A, push B)
Paso 3: [D] (pop B, push D)
Paso 4: [C] (pop D, push C)
Paso 5: [E] (pop C, push E)
Paso 6: [] (pop E)


7. BÚSQUEDA BFS (Breadth-First Search)
================================================================================

Grafo:
    A
   / \
  B   C
  |   |
  D   E

Orden BFS desde A: A -> B -> C -> D -> E
(Explora por niveles)

Queue (cola):
Paso 1: [A]
Paso 2: [B, C] (remove A, add B, C)
Paso 3: [C, D] (remove B, add D)
Paso 4: [D, E] (remove C, add E)
Paso 5: [E] (remove D)
Paso 6: [] (remove E)


8. CICLO EN GRAFO
================================================================================

Grafo con ciclo:
    A
   / \
  v   v
  B -> C
  ^    |
  |    v
  +----D

Ciclo encontrado: A -> B -> C -> D -> B -> ... (ciclo B-C-D-B)

Grafo sin ciclo (DAG):
    A
   / \
  v   v
  B   C
  |   |
  v   v
  D   E
  |   |
  v   v
  +-> F


9. COMPONENTES CONEXAS
================================================================================

Grafo desconectado:

    A --- B       C --- D
          |               |
          E               F

Componentes:
1. {A, B, E}
2. {C, D, F}


10. GRAFO COMPLETO K4
================================================================================

Todos los vertices conectados entre si.

      A
     /|\
    / | \
   B  |  C
   |\ | /|
   | \|/ |
   D--X--E

En K4 (4 vertices): cada vertice tiene grado 3
Total de aristas: 4 * 3 / 2 = 6


11. ÁRBOL DE EXPANSION MINIMA (MST)
================================================================================

Grafo original con pesos:
      1
    A---B
   /|\ /|
  2 |3 4|
  /| |/ \|
 C  D|E  5F

MST usando Prim/Kruskal:
      1
    A---B
   /   /
  2   4
  /   /
 C   E

Total peso: 1 + 2 + 4 = 7


12. CAMINO MAS CORTO (Dijkstra)
================================================================================

Grafo ponderado:
    1
A---B (1)
|\ /|
2|X|3
|/ \|
C---D (4)

Distancias desde A:
A: 0
B: 1 (A->B)
C: 2 (A->C)
D: 4 (A->B->D) o 5 (A->C->D)


13. LISTA DE ARISTAS
================================================================================

Para el grafo: A-B(5), A-C(3), B-D(4), C-D(2)

Representacion:
[
  (A, B, 5),
  (A, C, 3),
  (B, D, 4),
  (C, D, 2)
]

Uso: Algoritmos de MST (Kruskal), procesamiento eficiente


14. GRAFO BIPARTITO
================================================================================

Vertices divididos en dos conjuntos, aristas solo entre conjuntos:

Conjunto 1     Conjunto 2
  A              D
  |    \        / |
  |     X------/  |
  |    /        \ |
  B   /          E
   \ /
    X
   / \
  C    F

No hay aristas dentro de cada conjunto.


15. MATRIZ DE DISTANCIAS (Floyd-Warshall)
================================================================================

Grafo:
    1
  A-B-C
  |
  2
  |
  D

Matriz de distancias minimas:
    A  B  C  D
A [ 0  1  2  2 ]
B [ 1  0  1  3 ]
C [ 2  1  0  4 ]
D [ 2  3  4  0 ]

Cada celda [i][j] = distancia minima desde i hasta j
